# clear-cache
# 00:00时清理Linux页缓存

### 下载clear-cache.sh脚本并执行 

``` bash
curl -o clear-cache.sh -L https://raw.githubusercontent.com/Ccapton/myshell/master/clear-cache/clear-cache.sh && chmod +x clear-cache.sh && ./clear-cache.sh
```

### 执行clear-cache.sh脚本
``` bash
./clear-cache.sh
```

### 查看clear-cache进程
``` bash
ps aux|grep clear-cache
```
